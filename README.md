# TaskFlow - Gestión de Tareas Empresariales
TaskFlow es una solución tecnológica desarrollada para NovaTech Solutions que permite organizar y realizar el seguimiento de tareas dentro de un entorno corporativo. El sistema está diseñado como una aplicación de consola utilizando  C#.

## Funcionalidades
El sistema gestiona el ciclo de vida completo de una tarea a través de las siguientes funciones:
**Creación de Tareas**: Con título, descripción opcional y un responsable asignado.
**Listado de tareas**: Visualización de todas las tareas o filtrado entre los estados "Pendiente", "En progreso" y "Completada".
**Gestión de Estados**: Actualización del progreso de la tarea entre los estados definidos anteriormente.
**Reasignación**: Capacidad de cambiar el responsable de una tarea mediante su ID, registrando la fecha de modificación.
**Eliminación**: Borrado definitivo de tareas del sistema por su identificador.

## Interfaz de Usuario
Al ejecutar la aplicación, se dispone de las siguientes opciones:
1. **Crear tarea**.
2. **Listar tareas**.
3. **Actualizar estado**.
4. **Cambiar responsable**.
5. **Eliminar tarea**.
6. **Salir**.
